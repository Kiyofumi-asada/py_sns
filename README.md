# py_sns

## create venv flow

- venv

```
$ python3 -m venv venv_app
$ source venv_app/bin/activate
```

## add package flow

```
$ cd venv_app
$ pip install (modules)
$ pip freeze > requirements.txt
```

## start app flow

```
$ cd venv_app/py_sns
$ django-admin startproject py_sns
$ python manage.py runserver
```

http://127.0.0.1:8000/

## clone

```
$ source venv_app/bin/activate
$ pip install -r requirements.txt

```
